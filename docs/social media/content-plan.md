規劃社群貼文的切入點、順序與內容，逐步介紹網站功能與憲政議題。

需要規劃：
- 第一波貼文切入點（用什麼話題吸引注意？）
- 發文順序（先介紹哪個軌道？先談危機還是歷史？）
- 每篇貼文的核心訊息與 CTA（導流到哪個頁面？）
- 適合的社群平台（FB、IG、Threads、X？）
- 視覺素材方向（截圖、infographic、短影片？）
- 發文頻率與時程建議

## Stage Report

### 1. Analyze website content across all three tracks to identify shareable hooks — DONE

**Track 1 (Past — 憲法課本時光機)**
- Hook: 「課本 vs 現實」的視覺衝擊 — 左邊是課本上的民主理想，右邊是真實判決現場的黑白歷史照片
- Hook: 時間軸scrollytelling體驗 — 從1950s到2020s，每個年代都有「你以為理所當然的權利，其實是這樣來的」
- Hook: 結語的情緒轉折 — 「但是現在，憲庭載入中...」從驕傲轉為擔憂

**Track 2 (Present — 憲庭熱搜榜)**
- Hook: 懶人包（TL;DR）— 國會職權修法的重大判決，已有圖文整理
- Hook: 多元觀點對照 — 學者 vs 民間團體 vs 影片，不同來源的聲音並列
- Hook: 貓頭鷹法官的評論（owl_comment）— 用吉祥物口吻翻譯法律用語
- Hook: 重要時間表 — 從2019到未來的憲法法庭演變

**Track 3 (Future — 憲庭載入中)**
- Hook: 「Constitutional Emergency」紅色警示 — 15名大法官僅存5名運作
- Hook: 權益計算機 — 選擇你的身分標籤，看哪些待審案件影響你
- Hook: 瓶頸漏斗視覺化 — 案件只進不出的系統癱瘓
- Hook: 首頁數據即時預覽 — 待審案件數 + 大法官人數，一眼震撼

**Opinion Lazybag（大法官意見懶人包）**
- Hook: 散佈圖呈現論證光譜 — 不標籤人，而是理解不同法律論證角度

### 2. Propose first-wave posting angle — DONE

**第一波切入點：Track 3 的危機感**

理由：
1. 「15個大法官只剩5個在上班」是最容易在3秒內理解的事實，不需要法律知識
2. 危機感驅動分享 — 「你的權利正在排隊」比歷史回顧更有轉發動力
3. 網站的核心slogan「別讓你的權利已讀不回」與T3危機感完美對應
4. 權益計算機提供個人化互動 — 「看看你的權利排第幾號」是強CTA

先用危機感吸引注意，再用T1歷史感建立認同，最後用T2多元觀點深化理解。

### 3. Design posting sequence — DONE

**Phase 1：危機引爆（第1-3篇）— Track 3 Future**
1. 開場震撼彈：「15個大法官只剩5個」+ 權益計算機CTA
2. 身分連動：「你是___？你的案子排第___號」系列
3. 系統癱瘓解釋：為什麼會這樣？（立法院卡關 + 修法門檻）

**Phase 2：歷史縱深（第4-5篇）— Track 1 Past**
4. 「課本沒教你的事」：同婚/言論自由等案例的「課本vs現實」對照
5. 時光機體驗推廣：「從1950到2024，你的權利走了多遠？」

**Phase 3：深度理解（第6-7篇）— Track 2 Present + Opinion Lazybag**
6. 懶人包推廣：國會職權修法判決的完整整理
7. 大法官意見光譜：「不是選邊站，是理解為什麼」

**Phase 4：行動呼籲（第8篇）— 整合**
8. 串聯三軌：「過去的成就、現在的討論、未來的危機」完整敘事

### 5. Recommend platforms and format — DONE

**主力平台：**

| 平台 | 適合度 | 格式建議 | 理由 |
|------|-------|---------|------|
| **Threads** | 最高 | 文字為主 + 1張截圖/infographic | 台灣公共議題討論最活躍、分享門檻低、演算法友善新帳號 |
| **Instagram** | 高 | Carousel（輪播圖）5-7張 | 懶人包式內容最適合IG輪播，年輕受眾主力平台 |
| **Facebook** | 中高 | 長文 + 1張主視覺 | 30+歲受眾、社團轉發力強、適合深度內容 |
| **X (Twitter)** | 中 | Thread串文 + 數據截圖 | 政策圈/媒體人關注，但台灣用戶較少 |

**視覺素材方向：**
- **數據截圖**：直接截取網站的危機banner（15→5大法官）、瓶頸漏斗、權益計算機結果畫面
- **Infographic**：製作「課本vs現實」對照圖（左右分割，復古課本風 vs 新聞剪報風）
- **貓頭鷹吉祥物**：用owl mascot作為固定視覺識別，每篇貼文角落放小貓頭鷹
- **短影片（Phase 2+）**：螢幕錄製時光機scrollytelling體驗，15-30秒，配上字幕

**不建議一開始做的：**
- YouTube長影片（製作成本太高，先用截圖+文字驗證受眾反應）
- TikTok（與議題調性不符，除非有明確年輕族群策略）

### Summary

完成社群貼文內容規劃的 design 階段。分析了網站三軌道（Past時光機、Present熱搜榜、Future憲庭載入中）及意見懶人包的可分享素材，確定以 Track 3 危機感作為第一波切入點（「15個大法官只剩5個」），設計了8篇貼文的發布順序（危機→歷史→深度→行動），為每篇規劃了核心訊息與CTA導流目標頁面，建議以 Threads + Instagram 為主力平台搭配截圖/infographic/輪播圖格式，規劃每週二、五共4週的發文節奏，並撰寫了5篇可直接使用的貼文草稿。
