建立一份說明文件，記錄三個軌道的資料收集、更新、驗證流程：

- T1（時光機）：釋憲案資料來源、CSV 同步機制、更新頻率

3. 更新頻率
4. 更新步驟（SOP）
5. 驗證方式（verify 流程 or 人工確認）
6. 已知限制與風險
