### 2. Quiz Series Proposal

#### Quiz A:「你對憲法法庭爭議了解多少？」(必做題組)

- **來源：** `controversy-timeline.ts` + `future.ts` — 2024-2025 憲政風暴完整故事（時間軸事件 + 大法官人數危機）
- **憲法素養角度：** 了解台灣正在發生的憲政危機全貌——從國會擴權到法庭癱瘓
- **分享誘因：** 「你比 X% 的人更了解這場憲政風暴」— 時事性強，容易引發討論
- **敘事結構：** Q1 起因（國會擴權）→ Q2 反制（修法卡法庭）→ Q3 後果（人數危機）→ Q4 僵局（提名遭否決）→ Q5 結局（法庭自救判決）

**5 道題目：**

**Q1.** 2024年5月，國民黨與民眾黨聯手三讀通過「國會改革法案」，引發數萬人包圍立法院。這場群眾運動被稱為什麼？
- (A) 太陽花運動
- (B) 野百合運動
- **(C) 青鳥行動** ✓
- (D) 白色力量運動

> 教學點：這是 2014 年太陽花運動以來最大規模的國會相關群眾運動。憲法法庭隨後在 113 年憲判字第 9 號判決中宣告國會擴權法案多項核心條文（藐視國會罪、強制傳喚權等）違憲——但這個判決也激化了國會多數對法庭本身的反擊。

**Q2.** 國會擴權法案被宣告違憲後，立法院隨即修改《憲法訴訟法》反制。修法後，憲法法庭作成判決需要至少幾位大法官參與評議？
- (A) 8 人
- (B) 9 人
- **(C) 10 人** ✓
- (D) 15 人

> 教學點：這是「修法加卡人」雙殺策略的核心。表面上是「提高司法品質」，實際上搭配立法院拒絕行使大法官人事同意權，讓法庭永遠湊不到開庭人數。大法官法定員額 15 人，但 2024 年 10 月 31 日一口氣有 7 位大法官任期屆滿離任，僅剩 8 位——根本達不到 10 人門檻。

**Q3.** 賴清德總統兩次提名大法官人選送立法院行使同意權，結果如何？
- (A) 第一次通過、第二次被否決
- (B) 立法院尚未進行投票
- (C) 兩次都通過，但被提名人自行辭退
- **(D) 兩次都被立法院投票否決** ✓

> 教學點：第一次提名（2024 年 12 月 24 日）和第二次提名（2025 年 7 月 25 日）均遭國民黨與民眾黨控制的立法院否決。這使得憲法法庭始終無法補足人數，違憲審查持續停擺。若持續僵局，2027 年還有 4 位大法官任期屆滿（2019 梯次），屆時法庭將只剩 4 人；2031 年最後 4 位（2023 梯次）也將届滿，法庭將歸零。

**Q4.** 憲法法庭停擺期間，有多少件待審案件在排隊等待？
- (A) 約 50 件
- (B) 約 150 件
- (C) 約 300 件
- **(D) 約 473 件以上** ✓

> 教學點：這些不是抽象數字——涵蓋勞工加班工時、性侵害追訴時效、原住民權利、刑事被告人權等真實案件。每一件背後都有等不到正義的當事人。法庭停擺不只是政治角力，更是人民喪失釋憲救濟管道。

**Q5.** 2025 年 12 月，憲法法庭作成 114 年憲判字第 1 號判決，宣告《憲法訴訟法》修正案違憲。當時 8 位在任大法官中，最終有幾位參與作成這份判決？
- (A) 3 位
- **(B) 5 位** ✓
- (C) 8 位
- (D) 10 位

> 教學點：蔡宗珍、楊惠欽、朱富美三位大法官拒絕參與，最終僅 5 位大法官作成判決。判決確立了「立法權不得以修法方式實質廢除違憲審查」的憲政原則。張娟芬稱之為「台灣憲政制度的不自殺聲明」，並以波蘭國會用同樣手段癱瘓憲法法庭的前車之鑑作為對照。

### 4. Technical Approach

#### Page Route

```
/quiz/[quizId]          — 測驗答題頁面
/quiz/[quizId]/result   — 結果頁（帶 query params: ?score=4&total=5）
```

#### Data Structure

```typescript
interface QuizQuestion {
  id: string;
  question: string;
  options: { label: string; text: string }[];  // A, B, C, D
  correctAnswer: string;                        // "C"
  explanation: string;                          // 答對/答錯後的教學說明
}

interface QuizMeta {
  id: string;           // "controversy", "rights", "pending", "debate"
  title: string;        // 「你對憲法法庭爭議了解多少？」
  description: string;
  questions: QuizQuestion[];
  resultLevels: {
    minScore: number;
    title: string;      // 稱號
    description: string;
    owlMood: string;    // 貓頭鷹表情 variant
  }[];
  sourceRoute: string;  // 對應的網站內容頁路徑
}
```

#### Component Structure

```
src/app/quiz/[quizId]/page.tsx        — 主頁面（SSR quiz metadata + client interactivity）
src/app/quiz/[quizId]/result/page.tsx  — 結果頁（OG image generation）
src/components/quiz/QuizCard.tsx       — 單題卡片（選項、作答、解說）
src/components/quiz/QuizProgress.tsx   — 進度條
src/components/quiz/QuizResult.tsx     — 結果顯示 + 分享按鈕
src/data/quizzes/                      — 各測驗題組資料
```

#### Result Sharing Implementation

- **方案 A（推薦 MVP）：** 靜態結果頁 + OG meta tags。結果頁 URL 帶 `?score=4&total=5&quiz=controversy`，server 端根據 params 動態生成 OG image（使用 Next.js `ImageResponse` / `@vercel/og`）。分享時平台自動抓取 OG image。
- **方案 B（未來）：** Canvas 生成圖片供使用者下載，適合 IG Story。
